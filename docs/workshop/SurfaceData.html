Using SurfaceData Class

The SurfaceData Class takes in the constructor a surface and other parameters to analyze surfaces.

Code to implement SurfaceData Class

using LagaRhino;

  private void RunScript(Surface srf, int uCount, int vCount, ref object A, ref object B)
  {
    SurfaceData sd = new SurfaceData(srf, uCount, vCount);
    A = sd.SurfacePoints; //return the points
    B = sd.QuadPattern(); //return a squared pattern
  }

Method: PlaneSurfaceAxis()

This method takes a LineCurve and creates a flat surface from it. Internally the library transform the line into an axis.

Code to implement PlaneSurfaceAxis()

using LagaRhino;

  private void RunScript(Line ln, double height, ref object A)
  {
    A = LagaRhino.SurfaceData.PlaneSurfaceAxis(new LineCurve(ln), height);
  }
