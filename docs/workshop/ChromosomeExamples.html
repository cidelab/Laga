How to create Chromosomes

Chromosome Logic

There are several ways to create chromosomes with Laga. A chromosome is a Class that contains the DNA and the fitness value. The simple way is to generate DNA GenrDNA. and assign that data to the chromosome, which is like a container.

For example, to create a chromosome of non repeated integers you write this:

Chromosome<int> intChrShuffle = GenrDNA.Shuffle_Integer(0, 6);

The code above will create a chromosome with fitness 0, composed by the numbers between 0 and 6 (the last value is exclusive).

To assign a fitness value just do the following.

intChrShuffle.Fitness = 0.5;

Now that chromosome has a fitness of 0.5.

Code example

//Generating different chromosomes types
Chromosome<int> intChrShuffle = GenrDNA.Shuffle_Integer(0, 6);
Chromosome<int> intChrBin = GenrDNA.Binary_Integer(10);
Chromosome<char> chr = GenrDNA.Rand_Char(10, 50, 100);
Chromosome<char> chrBin = GenrDNA.Binary_Char(20);
Chromosome<float> fltChr = GenrDNA.Rand_Float(10, 2f, 90f);
Chromosome<double> dblChr = GenrDNA.Rand_Double(5, 0, 10);

// methods and examples to manipulate chromosomes
chr.Fitness = 0.5;
int size = chr.Count;
chr.InsertDNA(1, '3');
chr.Add('0');
har dnaAt = chr.GetDNA(3);

The Following code is to print the results in the console. Please note that we apply a FisherYates() algorithm to one of the chromosomes.

Console.WriteLine("Shuffle integers: " + intChrShuffle.ToString() + " Fitness: " + intChrShuffle.Fitness.ToString());
Console.WriteLine("Binary integers: " + intChrBin.ToString() + " Fitness: " + intChrBin.Fitness.ToString() + '\n');
Console.WriteLine("Random characters: " + chr.ToString() + " Fitness: " + chr.Fitness.ToString());
Console.WriteLine("Binary characters: " + chrBin.ToString() + " Fitness: " + chrBin.Fitness.ToString() + '\n');

Console.WriteLine("Random floats : " + fltChr.ToString() + " Fitness: " + fltChr.Fitness.ToString());
fltChr.Fisher_Yates();

Console.WriteLine("After Shuffle : " + fltChr.ToString() + " Fitness: " + fltChr.Fitness.ToString() + '\n');
Console.WriteLine("Random doubles: " + dblChr.ToString() + " Fitness: " + dblChr.Fitness.ToString() + '\n');
